# 运动装备电商项目网站

这是一个电商系统的后端服务，提供商品管理、用户认证、订单处理等功能。

## 技术栈

- **后端框架**：Spring Boot
- **数据库**：MySQL
- **前端**：Vue.js

## 功能特性

- 用户注册、登录与认证
- 商品浏览、搜索与分类
- 购物车管理
- 订单创建与支付
- 图片上传（头像、商品图片）

## 项目结构

- `src/main/java`：Java源代码
- `src/main/resources`：配置文件
- `uploads`：上传文件存储目录
- `frontend`：前端代码

## 如何运行

1. 克隆项目
2. 配置数据库连接
3. 运行后端服务
4. 启动前端应用

## 注意事项

- 确保`uploads`目录有正确的读写权限
- 生产环境部署时请修改默认配置
